
# versapy

##### <hr/>

### This package is [Versapy](https://github.com/Soli-64/VersaPy) front management module.

## <u> About the package </u>

#### ```versapy/api```: function ```invoke```, hook ```useSharedValue```
#### ```versapy/config```: function ```loadConfig```

#### See [Examples](https://github.com/Soli-64/VersaPy/tree/main/docs/examples/get_started.md) for further descriptions and use.

## <u> Usage </u>

### This package is auto-installed in your project by the cli [create-versapy](https://github.com/Soli-64/VersaPy/tree/main/cli) while creating your versapy project. <br/> See the [github repo](https://github.com/Soli-64/VersaPy) for more infos and exemples.
