
# versapy

##### <hr/>

### This package is the [Versapy](https://github.com/Soli-64/VersaPy) project utility js module.

## <u> About the package </u>

#### For now the only function is ```invoke```, which connects the front and the back in your versapy app.

## <u> Usage </u>

#### This package is auto-installed in your project by the cli [create-versapy](https://github.com/Soli-64/VersaPy/tree/main/cli) while creatingyour versapy app. <br/> See the [github repo](https://github.com/Soli-64/VersaPy) for more infos and exemples.
